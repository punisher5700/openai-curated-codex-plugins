# Superpowers Extension for Gemini CLI

This directory is a Gemini CLI extension that packages the Superpowers workflow skills.

## Install from this repo clone

```bash
bash gemini/install-superpowers-extension.sh
```

Or install directly from the extension path:

```bash
gemini extensions install /absolute/path/to/openai-curated/gemini/superpowers-extension
```

For live local development:

```bash
gemini extensions link /absolute/path/to/openai-curated/gemini/superpowers-extension
```

Restart Gemini CLI after install, enable, update, or unlink operations.
