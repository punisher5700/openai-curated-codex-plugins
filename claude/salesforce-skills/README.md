# Salesforce Skills for Claude Code

Claude-native plugin package for Salesforce development and architecture work.

Included:

- Salesforce architecture skills
- Apex, LWC, Flow, security, deployment, and integration skills
- low-token `salesforce-compact`
- audit-oriented `salesforce-reviewer`
- plugin-local MCP configuration through `.mcp.json`

This package is intended to be installed from the `openai-curated` Claude marketplace entry.
