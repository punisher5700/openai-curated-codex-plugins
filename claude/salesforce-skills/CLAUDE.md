# Salesforce Skills for Claude

This bundle provides Salesforce-focused skills for Claude Code.

Preferred workflow:

- use `salesforce-compact` for most implementation tasks
- use `salesforce-reviewer` to audit important answers and diffs
- use `salesforce-architect` for deeper design decisions

When correctness depends on repo or org evidence:

- inspect files first
- do not invent object or field names
- ask for the smallest missing artifact when evidence is incomplete
