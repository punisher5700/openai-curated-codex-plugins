# Salesforce Skills Templates

Reusable starter patterns for common Salesforce development tasks.

Included templates:

- `apex-trigger-handler.cls`
- `apex-service.cls`
- `apex-selector.cls`
- `apex-test.cls`
- `apex-queueable.cls`
- `apex-batch.cls`
- `apex-invocable.cls`
- `lwc/`
- `integration-service.cls`

These templates are intentionally generic and safe-by-default:

- bulk-aware
- separation-of-concerns friendly
- security-conscious
- deployable with adaptation
