# Superpowers for Claude

This bundle provides Superpowers workflows for Claude Code.

Use these when you want stronger engineering process, not domain-specific Salesforce behavior.

Recommended order:

- `brainstorming`
- `writing-plans`
- `executing-plans`
- `test-driven-development`
- `systematic-debugging`
- `verification-before-completion`

For larger feature work:

- `dispatching-parallel-agents`
- `subagent-driven-development`
- `finishing-a-development-branch`
